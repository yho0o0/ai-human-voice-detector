# AI vs Human Voice Detector

A Streamlit-based voice authenticity analysis app that estimates whether an uploaded WAV file is more likely AI-generated or human speech.

## Features
- Dynamic acoustic feature extraction
- MFCC, delta MFCC, delta-delta MFCC
- Spectral, energy, and pitch-based descriptors
- Ensemble machine learning model
- Interactive Streamlit web app

## Run locally
streamlit run app.py
